CSS Box Mode

Every element that can be displayed on web page is comproised of one or more rectangular boxes.CSS box model typically describes how these rectangular boxes are layed out on a web page. These boxes can have different properties and can interact with each other in different ways, but every box has a content area and optional surrounding padding, border, and margin areas
